# Fruit Slot Machine

A simple slot machine game built with JavaScript.

## Live Demo

https://alexandrabalinskaiya.github.io/lr10/

## Description

Fruit Slot Machine is a frontend practice project that simulates a slot machine.

The player has three attempts to spin the machine. During each spin, fruit symbols are randomly generated inside a 3×3 grid. The central row has an increased chance of containing matching symbols. If all three symbols in the middle row match, the player wins.

After all attempts are used, the game automatically resets.

## Features

* slot machine mechanics
* random fruit generation
* animated spinning effect
* win detection
* attempt counter
* automatic game reset
* image-based game interface

## Game Rules

1. Press the Spin button.
2. The slot machine generates random fruit symbols.
3. You have three attempts.
4. If all symbols in the center row match, you win.
5. If all attempts are used without a winning combination, the game resets.
